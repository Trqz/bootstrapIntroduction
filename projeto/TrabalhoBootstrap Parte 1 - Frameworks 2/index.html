Alerts

É um componente responsável por mostrar mensagens de feedback de acordo com ações típicas que um usuário possa realizar. Por exemplo:

  • Mensagem de confirmação de um evento;
  • Mensagem de erro;
  • Mensagem de atenção etc;

Alerta está disponível para textos. Utilize um estilo adequado de acordo com uma das oito classes contextuais disponíveis (por exemplo: alerta-sucesso).

Dipensando

Usando o plug-in JavaScript de alerta, é possível descartar qualquer alerta inline. Veja como:

  • Certifique-se de ter carregado o plug-in de alerta ou o arquivo compilado JavaScript do bootstrap.
  • Se você está criando nosso JavaScript a partir do código-fonte, ele exige o util.js. A versão compilada inclui isso.
  • Adicione um botão de rejeição e a classe alert-dismissible, que adiciona um preenchimento extra à direita do alerta e posiciona o botão fechar.
  • No botão dispensar, inclua o atributo data-dismiss = "alert", que aciona a funcionalidade JavaScript.
  • Certifique-se de usar o elemento button para o comportamento adequado em todos os dispositivos.
  • Para animar alertas ao descartá-los, certifique-se de adicionar as classes fade e show.
